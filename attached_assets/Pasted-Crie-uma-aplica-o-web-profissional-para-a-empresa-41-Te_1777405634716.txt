Crie uma aplicação web profissional para a empresa 41 Tech usando Next.js, React, TypeScript e Tailwind CSS.

Objetivo:
Criar uma landing page institucional + portfólio técnico/comercial da 41 Tech, setor de tecnologia do Grupo 41, demonstrando projetos, cases, experiências, equipe, stacks e soluções desenvolvidas em programação, automação, BI, dashboards, sistemas web, APIs, ERP, IA, n8n, Power BI, MySQL, Docker e EasyPanel.

Conceito:
A 41 Tech transforma processos manuais e operações complexas em sistemas, automações, dashboards e soluções digitais reais.

Crie um visual moderno, profissional e corporativo, misturando tecnologia, dados, automação e credibilidade comercial.

Stack obrigatória:
- Next.js com App Router
- React
- TypeScript
- Tailwind CSS
- Componentes reutilizáveis
- Layout responsivo
- Dark mode opcional
- Estrutura pronta para deploy via Docker/EasyPanel
- Banco MySQL preparado via Prisma ou Drizzle ORM
- Autenticação admin simples com senha criptografada

Projetos demo:
1. Hub Corporativo
Descrição: Plataforma interna para centralizar chamados, usuários, indicadores, permissões e ferramentas corporativas.

2. FinSight BI Financeiro
Descrição: Dashboard financeiro com DRE, indicadores, análise gerencial e integração com dados do ERP.

3. Automações n8n + Omie
Descrição: Fluxos automatizados para extração, tratamento e sincronização de dados via API.

4. ERP Interno
Descrição: Sistema web para centralizar processos de setores como fiscal, DP, BPO, comercial e gerência.

5. Integrações SprintHub
Descrição: Integrações entre CRM, automações comerciais, follow-ups e relatórios.

6. Dashboards Power BI
Descrição: Painéis executivos conectados ao MySQL para análise de dados operacionais e financeiros.
